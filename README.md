### whoami

GRC engineer, hacker, and builder. I spend my days running FedRAMP and NIST 800-53 assessments across AWS, GCP, and Azure — and my nights building open-source tools that make compliance less painful. I approach security work the same way I approach MMA training: technical, disciplined, and always looking for the edge.

Always open to collaborating. Feel free to reach out.

---

### what i work on

- 🔐 **Cloud Security Assessments** — FedRAMP, GovRAMP, DoD IL4/IL5 across AWS, GCP, and Azure
- 🤖 **GRC Automation** — Python tooling that replaces manual compliance workflows
- 💀 **Offensive Security** — red team, CTFs, HackTheBox
- 🛠️ **Open Source** — building tools practitioners actually use
